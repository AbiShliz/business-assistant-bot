# 🤖 Business Assistant Bot — Telegram бот для записи клиентов с AI на YandexGPT

Готовый Telegram-бот для малого бизнеса: салоны красоты, клиники, автосервисы, стоматологии, фитнес-клубы.  
Бот принимает заявки, отвечает на вопросы с помощью нейросети, отправляет уведомления администратору и хранит все данные в базе.

---

## 🚀 Возможности

| Функция | Описание |
|---------|----------|
| 📋 **Услуги и цены** | Автоматический показ прайс-листа из JSON |
| 📝 **Запись клиентов** | Интерактивный выбор услуги, даты и времени |
| 📞 **Сбор контактов** | Обязательное поле телефона для связи |
| 🧠 **AI-ассистент** | Ответы на любые вопросы через YandexGPT |
| 📅 **Мои записи** | Клиент видит свои активные записи |
| ❌ **Отмена записи** | Самостоятельная отмена через бота |
| ✍️ **Отзывы** | Сбор обратной связи с оценкой |
| 📍 **Контакты** | Адрес, телефон, часы работы |
| ❓ **FAQ** | Часто задаваемые вопросы (настраиваются) |
| 👤 **Админ-панель** | Статистика и просмотр записей |

---

## 🧠 Как работает AI

Бот использует **YandexGPT** (российская нейросеть) для:
- Ответов на нестандартные вопросы клиентов
- Вежливого и естественного общения
- Персонализированных приветствий

Никаких VPN, никаких блокировок — всё работает в РФ.

## ⚙️ Быстрый старт (для разработчика)

### 1. Получите токен бота
Напишите [@BotFather](https://t.me/BotFather), создайте нового бота и скопируйте токен.

### 2. Получите ключи YandexGPT
- Зарегистрируйтесь в [Yandex Cloud](https://cloud.yandex.ru)
- Создайте каталог → скопируйте **Folder ID**
- Создайте сервисный аккаунт с ролью `ai.languageModels.user`
- Создайте **API-ключ** и сохраните его
